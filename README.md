# Calculator

A simple calculator that supports PEMDAS, and some extra functionality.
+ Visit me online at https://thatguyjs.github.io/Calculator

Constants:
-
- PI: p
- E: e

Functions:
-
- Sqrt: q
- Abs: a
- Log: g
- Round: r
- Sin: s
- Cos: c
- Tan: t
- ASin: Shift + s
- ACos: Shift + c
- ATan: Shift + t

Known Bugs:
-
- Nested parenthesis can result in errors
